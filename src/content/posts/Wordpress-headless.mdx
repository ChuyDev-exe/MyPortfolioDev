# Configuracion de Wordpress modo headless

![WordPress Headless Architecture](https://static-academy.siteground.com/wp-content/uploads/sites/3/2024/07/Wordpress.jpg)

WordPress headless es una arquitectura donde WordPress actúa únicamente como un sistema de gestión de contenido (CMS) backend, mientras que el frontend se construye con tecnologías modernas como React, Vue, o Astro.
